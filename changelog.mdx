## Stay Up to Date

Paradoha gh evolves quickly to help you manage project documentation effectively. You receive automatic notifications for major releases through your dashboard at `https://dashboard.example.com`. Check this changelog regularly to see new features, improvements, and fixes that enhance your documentation workflow.

<Callout kind="tip">
  Subscribe to RSS feed at `/changelog/rss` or enable email notifications in your account settings to never miss an update.
</Callout>

## Recent Releases

Review the latest versions below. Each release includes detailed changes to keep your documentation space current and performant.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration editing, allowing multiple users to edit docs simultaneously with live cursors.
- Introduced version history with diff viewer, so you can compare changes across revisions easily.

## Improvements

- Enhanced search performance by `>50%` for large documentation spaces.
- Updated UI components with better dark mode support and accessibility improvements.

## Bug Fixes

- Fixed markdown rendering issues with nested lists and tables.
- Resolved export failures for PDFs containing images.

</Update>

## Upgrade Instructions

Follow these steps to upgrade Paradoha gh smoothly.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your documentation space via the dashboard.
  </Step>
  <Step title="Update Package" icon="package">
    
````bash
npm install paradoha-gh@latest
````

    
````bash
yarn add paradoha-gh@latest
````
  </Step>
  <Step title="Run Migrations" icon="git-branch">
    Execute database migrations:
    
````bash
npx paradoha-gh migrate
````
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Restart your Paradoha gh instance and verify at `https://dashboard.example.com`.
  </Step>
</Steps>
