## Overview

Create your first documentation project in a few steps.

## Steps

1. Sign up or log in at `https://paradoha.com`
2. Click "New Project" on your dashboard
3. Enter a project name and description
4. Click "Create" to initialize
5. Start adding pages with the editor

You're ready to organize your docs!
