## Overview

Start a new documentation project in Paradoha gh.

## Steps

<Steps>
  <Step title="Open Dashboard">
    Click "Dashboard" after login at `https://paradoha.com`.
  </Step>
  <Step title="Create Project">
    Click "New Project", enter a name and optional details, then click "Create".
  </Step>
  <Step title="Edit Documentation">
    Use the editor to add pages or upload content.
  </Step>
</Steps>
