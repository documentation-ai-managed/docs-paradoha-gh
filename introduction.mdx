## Overview

Paradoha gh empowers you to create, organize, and maintain project documentation in one centralized space. You build structured docs with version control, real-time collaboration, and powerful search capabilities. Whether you manage open-source projects or internal team knowledge bases, Paradoha gh simplifies your workflow and ensures your documentation stays up-to-date and accessible.

This platform integrates seamlessly with your GitHub repositories, automatically syncing changes and generating MDX-powered pages. You focus on content while Paradoha gh handles hosting, search indexing, and customization.

  Primary brand color: <code>#3B82F6</code>. Use this hex value for consistent theming across your docs, buttons, and logos.
</Callout>

## Quick Start

Get started with Paradoha gh in minutes.

<Steps>
  <Step title="Connect Repository" icon="github">
    Link your GitHub repo in the dashboard at <code>https://dashboard.example.com</code>.

    ```bash
    # Generate API token
    gh auth login --scopes repo
    ```
  </Step>
  <Step title="Configure Docs" icon="settings">
    Create your first MDX file in <code>/docs/introduction.mdx</code>.

````mdx
---
title: My Project
description: Overview of my project docs.
---
## Welcome
Start writing here.
````

  </Step>
  <Step title="Publish" icon="rocket">
    Push changes to trigger a live build.

    <CodeGroup tabs="GitHub CLI,npm">
      ```bash
      gh repo deploy --ref main
      ```
      ```bash
      npm run deploy
      ```
    </CodeGroup>
  </Step>
</Steps>
